# 建置筆記

# Git Config 優先級(高至低)
1. .git中的 config  (--local)
2. .gitconfig (--global)
3. git 安裝檔中的 git config (--system)
> 較低者，只有較高者未設置才會生效

# Git 記憶帳號
git config credential.helper store


# SassMap 用法
```scss
  $color: (
    pink: #ff6599;
  );
  h1 {
    color: map-get($color,pink)
  }
  @function color($color-name) {
    @return map-get($color, $color-name);
  }
  h2 {
    color: color(pink)
  }
```
